# Learning some pyqt

Ros has rqt but it lacks a lot of function and hard to use. 
So learn some pyqt to write GUI for ROS

##

```bash
conda install --channel https://conda.anaconda.org/conda-forge pyqt
conda install -c anaconda pyqt=4.11.4
```
